# Daneshjoosite (Django)

A simple Django project for managing students and filtering them by average score.

## Features
- Add / delete students
- View student profile
- Filter students by min_avg and max_avg

## Tech Stack
- Python
- Django
- SQLite

## Setup
1) Create virtualenv and install dependencies:
   pip install -r requirements.txt

2) Run migrations:
   python manage.py migrate

3) Start server:
   python manage.py runserver
